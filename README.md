# LiXin Library Seat Helper

上海立信会计金融学院 IC 空间座位预约油猴脚本。

作者：顾佳俊 23级金融科技 联系微信：AL-0729-zK

## 文件

- `lixin-library-seat-helper.user.js`：直接安装到 Tampermonkey 的用户脚本。

## 部署

1. 打开浏览器的 Tampermonkey 管理面板。
2. 新建脚本。
3. 删除默认内容。
4. 复制 `lixin-library-seat-helper.user.js` 的全部内容粘贴进去。
5. `Ctrl + S` 保存并启用脚本。
6. 打开 `https://kjyy.lixin.edu.cn/`，登录后进入任意“座位预约”页面。

## 使用

页面右下角会出现一个“座”按钮。

面板会在“目标座位号”上方显示登录状态，并展示距离下次检测还有多少秒。脚本每 10 秒发送一次只读检测请求：在座位预约页使用当前房间的真实座位查询接口，在首页或未进入房间时使用用户信息接口。检测会识别统一认证重定向、HTTP 登录失效状态和接口返回内容；不会点击左侧列表，也不会切换房间。

核心表单项有四个：

- 目标座位号：例如 `PDW3FA3001`
- 几点开始执行预约：普通输入框，格式为 `YYYY-MM-DD HH:mm:ss`；“现在”按钮会填入当前时间，“默认”按钮会填入今天 `22:30:00`
- 预约哪一天：下拉选择，支持今天到七天后，并会显示具体日期，例如 `明天（6月14日）`
- 预约时间段：默认 `08:00-22:30`，“重置”按钮可恢复默认时间段

面板会在座位号下方自动显示目标房间名和房间路由后缀，例如 `#/ic/seatPredetermine/100792125`。点击“提交任务”后，任务会出现在“当前任务”列表里，并实时显示距离执行的倒计时；点“取消”可删除。每个任务只按自己的日期时间执行一次，且预约日期会在创建任务时固定为当时下拉框显示的具体日期。

左键点击页面座位图上的任意座位圆点，会打开脚本面板并自动填入该座位号。占用的黄色座位也可以这样取到座位号。

## 注意

- 脚本需要浏览器打开、学校预约页面处于可访问状态时才会运行。
- 创建任务时会记录当前座位预约页面的房间路由；到点后会用该房间的真实座位查询接口查找座位，再向 `POST /ic-web/reserve` 提交预约，不依赖当前页面停留在哪个房间。
- 脚本不绕过验证码、登录、权限、预约规则或频率限制。
- 默认申请说明为空，写在脚本顶部的 `DEFAULT_APPLY_NOTE` 常量里。
- 到达设定时间后有 10 分钟宽限窗口；如果浏览器未打开、页面不可访问或错过窗口，任务会标记为已错过，不会事后补提交。
